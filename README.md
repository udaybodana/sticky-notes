# 📌 Sticky Notes

A clean, interactive digital notes board. Users can create, customize, and manage multiple sticky notes dynamically on the screen.

### ✨ Features
- Add new notes with real-time text editing.
- Customize individual note colors (Yellow, Pink, Blue, Green).
- Drag-and-drop capability for moving notes around the board.
- Delete individual notes instantly.

### 🛠️ Built With
- HTML5 & CSS3
- Vanilla JavaScript (DOM Creation, Event Listeners, Mouse/Drag Events)

### 💻 How to Run
Just open the `index.html` file in your browser.
